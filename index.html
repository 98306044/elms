elms by btopro

Drupal's e-Learning Management System

ELMS is a drupal distribution unlike any other. It has been built for instructional designers, by instructional designers with a major focus on usability, accessibility, and instructional flow. From a drupal perspective it's a different kind of distribution because it is an educational development framework that acts more as a wrapper and become many different kind of micro-site managing systems from the same code-base.

Dependencies

Drupal

Install

This repo is for the installation profile by itself, for a full-working distribution please visit ELMS on drupal.org
