Setup and Installation
1.) Download and add to sites/all/modules
2.) Enable module and adjust permission for "Clone Groups"
3.) Navigate to the group you wish to clone site-name.com/group-name and go to site-name.com/group-name/og_clone
4.) Select features from which content and from which roles you wish to clone, 
    set user who will be author of new nodes, and clone!
